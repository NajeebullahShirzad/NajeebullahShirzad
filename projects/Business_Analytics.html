Amazon Electronics Retail Data Analysis


Project Overview

This collaborative data science project analyzes a large-scale Amazon Electronics dataset (42K+ product listings) to understand how pricing, promotions, customer engagement, and category characteristics shape sales performance. Using extensive data cleaning, feature engineering, exploratory data analysis, and regression modeling, the project answers key managerial questions such as how discounts influence next-month sales and which products are most likely to become best sellers next quarter. The final analysis provides a data-driven framework for optimizing pricing strategy, promotional spending, inventory planning, and category prioritization in a highly competitive e-commerce environment.

Key Contributions

  • Data Engineering: Cleaned and transformed 42K raw records, reducing to 30K high-quality observations suitable for modeling.
  • Feature Engineering: Converted promotion features (sponsorship, coupon status) into binary variables for interpretability.
  • EDA: Conducted pricing distribution analysis, correlation studies, category-level performance comparisons, and engagement profiling.
  • Predictive Modeling: Built regression models to estimate next-month sales and forecast future best sellers using OLS.
  • Simulation: Developed scenario-based simulations to test the impact of discount changes, coupon activation, and sponsorship.
  • Interpretation: Generated business-ready insights for pricing strategy, promotional planning, and category management.

Skills Demonstrated

  • Data cleaning, preprocessing, and reduction
  • Exploratory data analysis (EDA)
  • Statistical modeling & OLS regression
  • Predictive analytics and scenario simulation
  • Feature engineering and dataset optimization
  • Category-level comparative analytics
  • Interpretation of correlations and promotional effectiveness
  • Technical reporting and insight communication

Technologies Used

  • Python – pandas, numpy, statsmodels, matplotlib
  • Jupyter Notebook – analysis environment
  • Kaggle dataset – Amazon Electronics 2025 dataset (42K items)
  • Google Drive – team collaboration & version control
  • Excel – supplementary EDA and visual checks
  • Regression modeling & simulation tools

Why This Project Matters

This project demonstrates the practical application of data science to real e-commerce challenges. By combining data engineering, statistical modeling, and business-oriented insight development, the analysis shows how retailers can move beyond intuition and use predictive analytics to improve pricing decisions, optimize promotional spending, and identify high-potential products. The workflow reflects a complete data science pipeline from raw data to actionable recommendations.
